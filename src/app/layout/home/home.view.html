Моё тестовое приложение было вдохновлено следующими статьями.

  • "Angular 1 Style Guide" by John Papa
  • Angular 1.5 - ES6/ES2015 Style Guide
  • Build an Angular 1.5 application with ES6 and Components

Данное приложение отражает мои идеи по созданию SPA на Angular-1 с использованием ES6 и компонентоного подхода. При его написании я прежде всего старался создать удобную модульную структуру, такую, чтобы впоследствии было просто заменить существующие модули либо добавить новые, т.е. легко дорабатывать и расширять приложение.

Компоненты являются "сердцем" приложения и я постарался их разделить на 2 типа:

  • 1. "layout-components", которые применяются в качестве HTML-шаблонов при написании роутов. Они должны быть использованы в качестве контейнеров для остальных компонентов
  • 2. "ui-components", которые представляют из себя блоки интерфейса, и при этом
    • реализуют одну чётко выраженную задачу,
    • инкапсулируют сложное поведение
    • предоставляют программисту-пользователю "robust and minimalistic API".

Благодарю за внимание к моему коду. :)
